Blender

Blender is a powerful open-source 3D software used for creating animated films and other computer animated visual effects. To learn more, please go here.

Some of the interesting features that Blender offers are the following: Photorealistic rendering, fast modeling, realistic materials, fast rigging, animation toolset, sculpting, and much more. To learn more about the features of Blender, go to their website. The link is attached here and at the top this page. Just select the title, Blender.

Credit: Information and image obtained from Blender's website.

ArcGIS

ArcGis is one of the most powerful mapping software available. This program allows you to input your own data, manipulate data, write your own code using python, allows geocoding, and is interactive. To watch a video on ArcGIS and to learn more, please select here.

To learn more, please go to the ArcGIS's website. You can select the title of this page, or click, here to go to their website.

Credit: Information provided and image received from ArcGIS website.

Mathematica

Mathematica is a powerful numeric computing program that is used by many in the scientific, mathematical, and engineering field. Able to do 2D and 3D geo visualization, data and matrix manipulation, and much more, Mathematica is useful for many in need of intense data crunching. To learn more about working in Mathematica, please take a look at this Lynda.com video by selecting here.
