Adobe Suites

Adobe offers many different applications that can assist you in your photo editing needs. InDesign, Illustrator, After Effects, Photoshop Lightroom are a few applicatinos that Adobe offers and that are available to you in the Port room. There are plenty of tutorials on Lynda.com that can help you learn how to navigate through the applications. If you want to check out the tutorials, just click,here.

GIMP

GIMP is a powerful image editor that crosses many platforms such as Linux, Windows, and OS X. In application, you can change the source code in this software which will improve your productivity. If you want to watch a tutorial on how to operate in GIMP, check out the tutorial on Lynda.com by clicking, here.

Orange Canvas

Orange Canvas is a data mining tool written with Python that provides data visualization. This is an interactive program that can help you interpret qualitative analysis. Orange Canvas has a visual programming front-end for explorative data analysis. If you want to learn more about working in Orange Canvas, select here.
